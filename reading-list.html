Reading List

As an engineer, I tend to read a lot of technical (and some non-technical) books throughout my life. Below is an always expanding list, in reverse chronological order, of the books I own and have either read or use for reference.

Security | Programming | Just for Fun | Everything Else

Security

  • The Hacker Playbook: Practical Guide To Penetration Testing by Peter Kim
  • Practical Malware Analysis: The Hands-On Guide to Dissecting Malicious Software by Michael Sikorski
  • Hacking Exposed 7: Network Security Secrets & Solutions by McClure, Scambray, and Kurtz

Programming

  • Design Patterns: Elements of Reusable Object-Oriented Software by Gamma, Helm, Johnson, and Vlissides
  • Ruby on Rails Tutorial by Michael Hartl
  • PHP and MySQL Web Development by Welling and Thomson
  • Accelerated C++: Practical Programming by Example by Koenig and Moo
  • Introduction to Java Programming by Y. Daniel Liang

Just for Fun

  • The Everything Store: Jeff Bezos and the Age of Amazon by Brad Stone
  • Without Their Permission: How the 21st Century Will Be Made, Not Managed by Alexis Ohanian
  • Hatching Twitter: A True Story of Money, Power, Friendship, and Betrayal by Nick Bilton

Everything Else

  • The Linux Command Line: A Complete Introduction by William E. Shotts Jr.

If you have any recommendations/suggestions, please let me know @andrewuf!
